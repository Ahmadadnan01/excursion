# Excursion Landing Page

This project is a simple landing page for the Excursion mobile app, built with HTML and CSS. It is designed for deployment on GitHub Pages.

## Structure
- `index.html`: Main landing page
- `resources/css/style.css`: Custom styles
- `resources/images/`: Placeholder for images (replace with your own)
- `resources/videos/`: Placeholder for videos (replace with your own)

## How to use
1. Replace placeholder images and videos in the resources folders with your own assets.
2. Edit the HTML and CSS as needed to match your product branding.
3. Deploy to GitHub Pages for a live demo.

## Deployment
Push to your GitHub repository and enable GitHub Pages in the repository settings. Your site will be available at `https://<username>.github.io/excursion/`.
